// GET STARTED //////////////////////////////////////////////

- From Unity's menu bar, select DOTween > DOTween Setup to setup the correct DOTween libraries based on your Unity version. If you don't see the DOTween menu, just refresh the menu bar by clicking on any of its menus. Repeat this step each time you update DOTween.
- In your code, add "using DG.Tweening" to each class where you want to use DOTween.
- You're ready to tween. Check out the links below for full documentation and license info.
